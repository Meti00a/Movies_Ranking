# KK1/Netflix(film och tv show)analys
A dataset based on tv shows and movies based on a Netflix dataset. 
-Created on Jupyter Notebook.

##dataset
**This dataset represents listings of movies and tv shows on Netflix**
sources from Kaggle: https://www.kaggle.com/datasets/shivamb/netflix-shows


## -How to run on Notebook:

Step 1: Download,unpack and extract the ZIP file that is provided from the keggle source.

step 2: Clone repo.

Step 3: Do a Pip install on Pandas, Numpy, matplotlib and seaborn.

step 4: add the file in the data mapp.

step 5: open Python.ipynb in vscode through jupyter.

step 6: run all cells from up and downwards.
